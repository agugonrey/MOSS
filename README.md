### Installing and loading MOSS from CRAN.

```
install.packages("MOSS")
library("MOSS")
```
### Installing and loading MOSS from GitHub.

```
if (require("remotes") == FALSE) install.packages("remotes")
install_github("agugonrey/MOSS")
library("MOSS")
```
